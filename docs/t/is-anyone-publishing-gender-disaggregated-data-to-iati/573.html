Community Zone
andylolz (Andy Lulham) September 20, 2016, 3:14pm #1

Is anyone publishing gender disaggregated data to IATI? Any links to samples or pointers to organisations doing this appreciated.

bjwebb (Ben Webb) September 20, 2016, 4:16pm #2

Looking at http://dashboard.iatistandard.org/elements.html it looks to be only unitedstates who are publishing the dimension element, and even then it looks to be only one activity http://datastore.iatistandard.org/api/1/access/activity.xml?iati-identifier=US-18-GE2-2750BXCOMD-TEP from this file https://iatiregistry.org/dataset/unitedstates-ge2

Looks like they are disaggregating by gender though.
