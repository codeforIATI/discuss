[Added] New Organisation Identifier: XI-IATI-SGJ

Standard Management Non-Core Codelist Amendments
IATI-techteam (IATI Technical Team) March 1, 2019, 10:36am #1

The following addition to the IATI Organisation Identifier list has been proposed for Somali Gender Justice. The request was reviewed by org-id.guide for the most appropriate identifier, and it was concluded that no alternative organisation identifier is currently available.

Code: XI-IATI-SGJ
Name: Somali Gender Justice
Description: Somali Gender Justice is a small NGO based in Garowe, Puntland, Somalia whose goal is to support gender equality, women’s rights, and community transformation.
URL: https://www.facebook.com/SomaliGenderJustice/

If no objection is received, this XI-IATI code will be approved and added in a week.
