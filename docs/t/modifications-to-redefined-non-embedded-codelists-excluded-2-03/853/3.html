Modifications to redefined non-embedded codelists (excluded 2.03)

Standard Management 2.03 Decimal Upgrade Proposals
activity_standard, 2-03_codelists
IATI-techteam (IATI Technical Team) June 16, 2017, 3:37pm #1

This proposal is part of the 2.03 upgrade process, please comment by replying below.

Standard
Activity and Organisation

**Schema Object** None

**Type of Change** None

**Issue** A number of proposals have been logged to modify codelists that will now become non-embedded. These changes can now be made through the [non-embedded consultation process](http://iatistandard.org/202/codelists/codelist-management/) and can be removed from the upgrade agenda.

**Proposal** Move the following proposals out of the upgrade process.

  • [Document Category](https://discuss.codeforiati.org/t/additions-to-document-category-codelist/782)
  • [Activity Scope](https://discuss.codeforiati.org/t/changes-needed-to-activity-scope-codelist/527)
  • [User-defined Vocabularies](http://iatistandard.org/202/guidance/how-to-publish/establish-publishing-policies/)
  • [BudgetIdentifierVocabulary](https://docs.google.com/document/d/1kOIcA2A_eDfP8pnoLTFHbXvRdJdwJn81R2CwdQcrqQc/edit)
**Standards Day** No information available

**Links** [https://discuss.codeforiati.org/t/sectors-sector-mappings/173/13](https://discuss.codeforiati.org/t/sectors-sector-mappings/173/13)
