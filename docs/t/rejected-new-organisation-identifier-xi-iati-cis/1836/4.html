[Rejected] New Organisation Identifier: XI-IATI-CIS

Standard Management Non-Core Codelist Amendments
IATI-techteam (IATI Technical Team) September 23, 2019, 12:53pm #1

The following addition to the IATI Organisation identifier list has been proposed for CARE International Foundation - Switzerland (registered in Sudan with the Humanitarian Aid Commission).

The request was reviewed by org-id.guide for the most appropriate identifier, and it was concluded that no alternative organisation identifier is currently available.

Code: XI-IATI-CIS
Name: CARE International Foundation - Switzerland
Description: Working on the humanitarian and live saving activities in Sudan
URL: https://www.care.org/country/sudan

If no objection is received this XI-IATI code will be approved and added in a week.

IATI-techteam (IATI Technical Team) September 6, 2019, 9:31am #3

Hi Matt,

They could not use a Swiss identifier because CARE International Switzerland are not actually registered there, and mainly operate in Sudan.

The reg no. that you have provided is actually for CARE as a whole, so would not be suitable.

However, we have since found that CARE International Switzerland in Sudan have been provided an ID by UN OCHA (http://org-id.guide/list/XM-OCHA) so we will instead proceed with using the UN agency code instead.
