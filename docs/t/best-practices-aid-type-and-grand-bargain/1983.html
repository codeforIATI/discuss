Best Practices - Aid Type and Grand Bargain

Community Zone Using IATI Data
Michelle_IOM (Michelle Levesque) July 2, 2020, 5:12pm #1

Hello Everyone,

Hoping to crowd source some ideas on best practices. Apologies if this has been discussed before but I didn’t see it in previous discussions but admittedly I’m crap at finding old threads. If I missed it don’t hesitate to point me to it.

There are AidType vocabulary for both OECD and Grand Bargain (GB) and from what I can tell you can code the same activity (default level) or transaction (transaction level) against both.

Assuming that a publisher is:

  1. receiving money from a donor
  2. is not passing it on to an implementing partner
  3. coding at the transaction not activity level

Questions, is it correct to assume:

  1. that codes 1-3 would only be used for incoming commitment (transaction type 11) or incoming funds (transaction type 1)
  2. code 4 would be for expenditures?
  3. is it best practice to code both transaction types 1 and 11 with aid types 1-3 or would it only be for the incoming commitment?

Rather than replicating AidTypes in two different transaction types, it could simply be coded at the activity level. But if these 3 vocabularies are only related to two transaction types won’t that mess up assumptions on the disbursement (3) and Expenditure (4) transaction types?

In looking at the list of GB signatories, only a handful actually code to the GB aid type vocabulary 2-4 so perhaps @stolk, @pelleaardema, @stevieflow can chime in here as they work for/with a few of them.

Thanks in advance.
Michelle

IATI-techteam (IATI Technical Team) July 14, 2020, 9:30am #2

Hi @Michelle_IOM Thank you for opening this discussion on best practice. Currently, we don’t have a specific recommendations on which transaction types are expected to be seen for each of the aid-type vocabularies you have referred to above.

This is indeed something that we want to incorporate into the humanitarian guidance consultation that we plan to have in the upcoming months, and it will indeed be good to have Grand Bargain signatories involved.

There are still very few publishers using vocabularies 2, 3 and 4. You can see which organisations have already used the codes activity level and/or transaction level. It will be great to hear from those already publishing as well as others that plan on using them shortly.
