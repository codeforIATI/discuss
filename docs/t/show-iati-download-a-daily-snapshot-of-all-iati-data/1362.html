IATI Data Dump

Download a daily snapshot of all IATI data (and metadata) from the IATI Registry.

Lots of IATI tools need to download all IATI data, and they each have their own code to do it. Even on a fast connection, it can take a few hours. Swapping in IATI data dump as a microservice would significantly speed up (for instance) the IATI Dashboard updates (which no longer run daily because of how long it takes).
