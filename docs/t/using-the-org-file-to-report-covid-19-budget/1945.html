Using the org file to report Covid-19 budget?

Consultations COVID-19
Melinda (Melinda Cuzner) April 29, 2020, 10:17am #1

Has anyone considered using the org-file to report how much of one‚Äôs budget is allocatied/redirected to Covid-19? The reason I ask is that some organizations can say of their budget is expected to go to Covid-19 but are unable to specify activity data. With other words, make the org file more of a narrative of an organizations overall budget instead of focusing on the forward looking aggregated budget. Last summer @stevieflow brought up the question if the org file is used (under topic: Does anyone use the Organisation file?). Maybe we could think wider about the use cases.

However, the org file specifies where the money is going (org or country) but doesn‚Äôt specify the purpose as far as I can see. So this may not at all be reasonable.
