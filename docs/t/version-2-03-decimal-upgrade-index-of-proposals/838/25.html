Agenda for 5 September call on Humanitarian Proposals

  • Introductions
  • Confirm submission of request to DAC WP-STAT to add ‘Cash transfer programming’ to Aid Type codelist
  • Consider proposal to add activity-based pledges as a transaction type
  • Consider proposal to allow for multiple aid type vocabularies to encompass, inter alia, humanitarian earmarking
  • Consider proposal to add related-transaction element to establish connections between transactions in the same activity
  • Consider proposal regarding additions to the Organisation Type codelist
