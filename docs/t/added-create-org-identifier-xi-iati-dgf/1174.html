[Added] Create Org. Identifier XI-IATI-DGF

Standard Management Non-Core Codelist Amendments
OJ_ (Ole Jacob Hjøllund) February 5, 2018, 3:21pm #1

Enabling IATI-publication from the programme management unit, the Development Governance Facility (Uganda) is proposed to be included in the XI-IATI list of identifiers.

Code: XI-IATI-DGF

Name: Democratic Governance Facility

Description: DGF supports state and non state partners to strengthen democratisation, protect human rights, improve access to justice and enhance accountability in Uganda. DGF’s work is built upon the principles and values enshrined in Uganda’s 1995 Constitution and reiterated in its current National Development Plan.

URL: https://www.dgf.ug/

Process for adding `XI-IATI` codes
IATI-techteam (IATI Technical Team) February 5, 2018, 10:55am #2

The IATI tech team approves this addition to the XI-IATI Organisation identifier codelist.
